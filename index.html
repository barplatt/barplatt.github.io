Jeet Grid Photo Gallery

Found a great grid system (sass or stylus) called Jeet so I though I'd put it to the test and quickly draw up some examples for a photo gallery. If you can't use flexbox in your next project give this lightweight, highly flexible system a try. Under "blog" is a spin-off aimed strictly at text blogging. (Update: Jeet is currently unmaintained but it's solid as-is). Fork it on git

The nut behind the scenes

Often the daily grind of projects calls for a conservative use of “new” technology so I just wanted to use this site to show some of the “other stuff” I have been working on. I built this site, for instance, using Flexbox Grid, mostly to try it out. New tech gets mainstream quickly so staying curious about the trends in both design and tech is imperative. Soon enough, floating an element might be regarded as quant as the use of tables for layout. The pdf browser of my dreams has yet to arrive, however. App developers—you’re welcome. Thank me later ;-)

Update: I probably won't use the Flexbox Grid again anytime soon. Grids are great for throwing something together quickly and for teams. This particular one, however, fell short. The cards, for one (with a button fixed to the bottom) were not possible to create with the grid.

Curious note: the “be BOLD” logo was just intended as a placeholder but since it fits well with my shaved head, outlook on design and life in general, I just kept it there.
